Quick Reference

Characters

  • . Any character
  • \d Digit [0-9]
  • \w Word char
  • \s Whitespace

Quantifiers

  • * Zero or more
  • + One or more
  • ? Zero or one
  • {n,m} Between n and m

Anchors

  • ^ Start of string
  • $ End of string
  • \b Word boundary

Groups

  • (abc) Capturing
  • (?:abc) Non-capturing
  • (?<name>abc) Named
