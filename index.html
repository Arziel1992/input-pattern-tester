Pattern Cheat Sheet

Email Validation

[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}$

Matches standard email format like: example@domain.com

Phone Number (10 digits)

[0-9]{10}

Matches 10 digit numbers without spaces or dashes

Phone with Format

\(\d{3}\) \d{3}-\d{4}

Matches phone format: (123) 456-7890

Date (DD/MM/YYYY)

(0[1-9]|[12][0-9]|3[01])/(0[1-9]|1[012])/\d{4}

Matches dates in format: 25/12/2023

Date (YYYY-MM-DD)

\d{4}-(0[1-9]|1[012])-(0[1-9]|[12][0-9]|3[01])

Matches ISO date format: 2023-12-25

Time (24 hour)

([01][0-9]|2[0-3]):[0-5][0-9]

Matches 24-hour time format: 14:30

Time (12 hour)

(0[1-9]|1[0-2]):[0-5][0-9] (AM|PM)

Matches 12-hour format: 02:30 PM

URL

https?://[a-z0-9.-]+\.[a-z]{2,}

Matches URLs like: https://example.com

Password (Strong)

(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,}

Requires at least 8 chars with lowercase, uppercase and number

Username

[a-zA-Z0-9_-]{3,16}

3-16 characters, letters, numbers, underscore, hyphen

Hex Color Code

#[a-fA-F0-9]{6}

Matches hex color codes like: #FF5733

IP Address

((25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\.){3}(25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)

Matches IP addresses like: 192.168.1.1

Regex Quick Reference

Character Classes

  • . - Any character except newline
  • \d - Digit (0-9)
  • \w - Word character (a-z, A-Z, 0-9, _)
  • \s - Whitespace (space, tab, newline)
  • [abc] - Any of the characters a, b, or c
  • [^abc] - Any character except a, b, or c
  • [a-z] - Any character from a to z

Quantifiers

  • * - 0 or more occurrences
  • + - 1 or more occurrences
  • ? - 0 or 1 occurrence
  • {n} - Exactly n occurrences
  • {n,} - n or more occurrences
  • {n,m} - Between n and m occurrences

Anchors & Boundaries

  • ^ - Start of string
  • $ - End of string
  • \b - Word boundary

Groups & Alternation

  • (abc) - Capture group
  • a|b - Match a or b
  • (?:abc) - Non-capturing group
